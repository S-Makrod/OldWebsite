The Development Journey

I made this we application because I wanted to improve my Web Development skills. To do this I had to research about PHP, and MySQL. I took my time learning about SQL and HTML injection, CRUD functionality, Model-View-Controller design, and how to use PHP to interact with an SQL server.

Once I learned all these topics I was able to develop this application. You can sign in to the web application and then create, read, update, and delete automobiles. Each automobile has a make, model, year and mileage and there is data validation before inserting the data into the database.

I used htmlentities() and PDO prepare statements to avoid SQL and HTML injection.

The following features were implemented:

  • Login Page
  • Create: You can create automobiles through the "Add New Entry" feature
  • Read: A table is shown for viewing the data (only if you are signed in)
  • Update: Click the edit link to update an entry (only if you are signed in)
  • Delete: Click the delete link to delete an entry (only if you are signed in)
