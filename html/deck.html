Deck GUI

The Development Journey

I began this project in May 2020. When I started designing my project I decided on the commands I wanted to carry out. I decided on the following operations, note that the operations deal and search are zero based so the first card is at index zero:

  1. Shuffle: This randomizes the deck of cards
  2. Sort Rank: This sorts the cards by rank by using quicksort
  3. Sort Suit: This sorts the cards by suit using selection sort
  4. Deal: This will deal card number x where x is choosen through the combo box next to the deal option
  5. Add: This will add a card to the end of the deck, the card is specified by the two combo boxes to the right
  6. Search: This will hide all cards that do not match the card you are looking for and output the positions of the cards on the screen

After I decided on these operations I began to implements my code. I developed a Card class and Deck class to promote encapsulation and then I implemented the Java Swing GUI. After I completed my code I tested my program for any bugs and this resulted in the program you see above.
