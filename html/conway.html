Conway's Game of Life

The Development Journey

I made this project in May 2020 when I first began using Java Swing. A good introduction to Conway's Game of Life can be found here: Conway's Game of Life Intro. When I learned about Conway's Game of Life I found out that there are special patterns hence I made preloaded patterns for the user. I also did not want to limit the user so I decided to create a randomize feature and a drag to draw feature so that user can populate the map theirself. I also needed to develop a clear and erase feature for the user so that they could properly control the simulation. Finally I decided to implement a save feature so that the user can save their pattern in a textfile for later. I would like to point out that in the image carousel and video the note is incorrect. It should say "Note: Do not add a file extension as it is automatically done for you", the code on my Github has been updated to reflect this.

After deciding on my plan I began to create the Java Swing interface and the Colony class so that I could implement the features I wanted to create. I then debugged my code for any logic errors resulting in the program you see above.
